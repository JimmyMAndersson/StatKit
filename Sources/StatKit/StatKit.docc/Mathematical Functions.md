# Mathematical Functions

Compute special mathematical functions used in probability and statistics.

## Overview

StatKit exposes a set of special mathematical functions that are commonly used in probability theory and statistics.

## Topics

### Binomial Coefficients

- ``choose(n:k:)``

### Beta Functions

- ``beta(alpha:beta:logarithmic:)``
- ``regularizedIncompleteBeta(x:alpha:beta:)``

### Gamma Functions

- ``incompleteGamma(lower:upper:alpha:logarithmic:)``
- ``incompleteGamma(lower:upper:alpha:)``
- ``regularizedIncompleteGamma(lower:upper:alpha:logarithmic:)``
